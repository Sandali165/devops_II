# DevOps II Project

\# DevOps II Project



\## Description

This is a Spring Boot based DevOps project demonstrating CI/CD pipeline, Docker integration, and deployment.



\## Technologies Used

\- Java

\- Spring Boot

\- Maven

\- Docker



\## Features

\- REST API implementation

\- Environment-based configuration

\- Containerization using Docker



\## How to Run

1\. Clone the repository

2\. Run: mvn clean install

3\. Run: java -jar target/\*.jar
